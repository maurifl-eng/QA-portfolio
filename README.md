# Portfolio QA - Control de Calidad y Automatización

Este es mi portfolio de proyectos para mejorar mis habilidades como QA, tanto manual como de automatización.

## Propósito

- Mostrar proyectos de prueba manual y automatización.
- Usar herramientas como Selenium WebDriver, Java, TestNG.
- Documentar mi evolución con buenas prácticas, reportes de bugs y estrategias de prueba.

## Áreas de interés

| Tipo                     | Tecnologías / Herramientas                                       |
| ------------------------ | ---------------------------------------------------------------- |
| **Manual Testing**       | Casos de prueba, reportes de bugs (Jira), pruebas exploratorias  |
| **Automation (Web)**     | Selenium WebDriver, Java, TestNG, Maven, Page Object Model (POM) |
| **Control de versiones** | Git, GitHub                                                      |

## Proyectos destacados

### 1. Pruebas manuales
- Login y flujo de compra.
- Plan de pruebas, casos positivos/negativos, reporte de bugs.

### 2. Automatización con Selenium
- Suite para Saucedemo.
- Java + Selenium + TestNG + POM.
- Capturas en fallos, reporte HTML.
- Ejecutar: `mvn clean test`

### 3. API Testing (en desarrollo)
- Postman para endpoints REST.

## Cómo usar esto
1. Explorá cada carpeta.
2. Leé los README internos.
3. Para correr los tests: Java 11, Maven, ChromeDriver.
